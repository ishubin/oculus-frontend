Oculus Frontend Introduction

  • Project Management You can manage multiple test projects, split tests to sub-projects, manage project builds and more
    Screenshots
  • Test-Case Editor You can also write test-cases with WYSIWYG text editor or plain text editor
    Screenshots
  • Grid Not only Oculus Frontend has a Grid Server embedded but it has an extensive UI for managing/running tests in Grid
    Screenshots
  • Reporting System Oculus Frontend has an Advanced Report Search functionality. And for each test report there is a handy report page with tree navigation
    Screenshots
  • Advanced Suite Editor You can implement quite a complex logic in your test suites. Oculus Frontend offers a suite editor for Grid tasks where you can provide a dependency between tests
    Screenshots
  • Issue tracking Oculus Frontend provides a simple support for tracking issues and also allows you to link your test runs with issues.
    Screenshots
  • Customization Engine If you do not like the given set of control over your test projects you can create your own settings for different units: Tests, Issues, Sub-Projects, Builds.
    Screenshots
