Installation Guide

Download binary package from Download Section. Extract it anywhere on your file system.

Actually the only thing you need to prepare is the Database. On Linux you can just just execute ./install.sh script and follow futher instructions. Or you can execute a database dump in oculus-db-init.sql file. 

mysql --verbose --user=root --password=root123 oculus < oculus-db-init.sql

Configure and Run Frontend

All configurations can be modified in oculus.properties file. 

# Used for authorization check of all API requests.
# API request must include either 'Auth-Token' HTTP header,
# either 'auth_token' HTTP parameter
# In case if token is not specified or is empty there
# will be no authorization need and all requests can
# be provided without 'Auth-Token' header
api.token =

# Network port for running oculus frontend
oculus.server.port = 8080

# Internal file storage
data.folder = /opt/ofe-data

# Mail sending settings
mail.smtp.host = localhost
mail.smtp.port = 25
mail.sender.name = OCULUS


# Used for generating a link in offline reports
oculus.offline.report.server.url = oculus-frontend:8080

# Specifies if a Grid Server should be run embedded in oculus frontend.
grid.embedded = true

# Grid Server hostname.
# In case if grid.embedded = true this hostname should be 'localhost'
grid.server.host = localhost

# Remote name for Grid Server
grid.server.name = grid

# Network port for running and connecting to Grid Server
grid.server.port = 8081

# Local project storage path for Grid Server
grid.server.storage = /opt/ofe-data/storage-server


# MySQL Database connection settings
db.host = localhost
db.port = 3306
db.scheme = oculus
db.username = root
db.password = root123

On Linux you can run ./run.sh scripts or just run the following command from oculus-frontend directory. 

java -jar oculus-frontend.jar

Integrate with Grid

By default Grid Server is already embedded into Oculus Fronted. So the only thing you need to do is to setup Grid Agents and point them to the embedded Grid Server. For configuring Grid Agents please read this manual.
